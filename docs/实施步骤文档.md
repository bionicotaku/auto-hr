## 4. 当前系统真值

### 4.1 Job

- Job 保存走 `job_finalize` analysis run
- 保存成功后更新当前生效内容
- `/jobs` 不展示 `draft`

### 4.2 Candidate

- Candidate 分析走 `candidate_import` analysis run
- Candidate 只在资料有效且分析成功时入库
- 硬门槛只影响默认标签与展示，不阻断入库
- Candidate 原始文件只保留原 PDF 和访问链接

### 4.3 前端

- Job 详情页 Candidate 列表筛选是页面本地状态
- 不再通过 URL 驱动
